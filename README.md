# 💎 Gemini Vision Studio

**Gemini Vision Studio** adalah platform pembuatan gambar berbasis AI yang memanfaatkan model **Gemini 2.5/3 Flash Image (Nano Banana)**. Aplikasi ini memungkinkan pengguna untuk menghasilkan karya seni visual hanya melalui deskripsi teks secara instan.

## ✨ Fitur Utama
* **AI Generation**: Mengubah prompt teks menjadi gambar berkualitas tinggi.
* **Timestamp AM/PM**: Setiap gambar di galeri dilengkapi dengan catatan waktu pembuatan (AM/PM).
* **Smart Download**: Fitur unduh gambar dengan nama file otomatis yang menyertakan penanda waktu.
* **Interactive Gallery**: Menyimpan riwayat gambar selama sesi berlangsung untuk dilihat kembali.
* **Regenerate & Retry**: Tombol untuk memicu pembuatan ulang gambar dengan cepat jika hasil sebelumnya kurang memuaskan.

## 🛠️ Tech Stack
* **Framework**: [Next.js](https://nextjs.org/) (App Router).
* **AI Engine**: Google Generative AI (Gemini API).
* **Styling**: Inline CSS / Tailwind-ready.
* **Language**: JavaScript (React).
